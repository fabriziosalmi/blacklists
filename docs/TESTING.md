# Testing Guide

This guide provides instructions for testing the blacklists project.

## Unit Tests

To run the unit tests, execute the following command:

```bash
pytest
```

## Integration Tests

To run the integration tests, execute the following command:

```bash
pytest --integration
```

## End-to-End Tests

To run the end-to-end tests, execute the following command:

```bash
pytest --e2e
```
